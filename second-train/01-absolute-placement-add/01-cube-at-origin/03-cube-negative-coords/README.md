# Cube at Negative Coordinates

## Description
Test cube placement at negative world coordinates.

## Task
- Keep the existing geometry unchanged
- Add the new primitive as specified
- Use absolute world coordinates

## Key Learning Points
- Negative axis handling, corner semantics with negative values
- Absolute coordinate interpretation
- Additive union modeling

## Expected Result
The output contains the original sphere plus one new cube at corner [-5, -3, -2] with size [3, 3, 3].
