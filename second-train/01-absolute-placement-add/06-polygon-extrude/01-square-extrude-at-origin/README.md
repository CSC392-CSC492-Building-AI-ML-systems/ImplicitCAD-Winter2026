# Extruded Square at Origin

## Description
Test 2D polygon extrusion to create 3D geometry.

## Task
- Keep the existing geometry unchanged
- Add the new primitive as specified
- Use absolute world coordinates

## Key Learning Points
- 2D-to-3D extrusion, polygon geometry, height parameter
- Absolute coordinate interpretation
- Additive union modeling

## Expected Result
The output contains the original sphere plus one extruded square at origin.
