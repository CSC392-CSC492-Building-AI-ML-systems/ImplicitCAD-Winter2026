# Sphere R At XYZ Test

## Description
This test validates direct absolute center placement for a sphere.

## Task
- Keep the existing geometry unchanged
- Add a sphere with radius `2.5`
- Place its center at absolute coordinate `[2,-3,4]`

## Key Learning Points
- Direct center placement behavior of `sphere()` under translation
- Correct usage of decimal-valued radius
- Absolute positioning with signed coordinates

## Expected Result
The output contains the original object plus one sphere centered at `[2,-3,4]` with `r=2.5`.
