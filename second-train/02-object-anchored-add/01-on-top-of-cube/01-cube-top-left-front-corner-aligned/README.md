# Cube Top-Left-Front Corner Aligned Test

## Description
This test validates the model's ability to add a cube on top of an existing cube with corner alignment rather than center alignment.

## Task
- Start with a base cube at origin [0,0,0] with size [4,4,4]
- Add a second cube of the same size positioned so its bottom-left-front corner aligns with the top-left-front corner of the base cube
- The second cube should be at position [0,0,4]

## Key Learning Points
- Understanding spatial relationships between objects
- Corner-based positioning vs center-based positioning
- Vertical stacking with alignment constraints

## Expected Result
Cube with another cube on top, corner-aligned: a structure with two cubes stacked vertically, sharing the same x,y coordinates but with the second cube elevated by the height of the first cube.
