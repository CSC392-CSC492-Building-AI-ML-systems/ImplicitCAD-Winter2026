# Cube Centered on Cube Top Face Test

## Description
This test validates the model's ability to add a cube on top of an existing cube with center alignment on the top face.

## Task
- Start with a base cube at origin [0,0,0] with size [4,4,4]
- Add a second cube of the same size positioned so its center aligns with the center of the base cube's top face
- The second cube should be at position [0,0,4]
- Both cubes share the same x,y bounds when viewed from above

## Key Learning Points
- Understanding face-centered positioning
- Center-based alignment concepts
- Vertical stacking with centered alignment
- Spatial reasoning about cube centers vs corners

## Expected Result
Two cubes stacked vertically with their centers aligned, creating a uniform column when viewed from the side.
