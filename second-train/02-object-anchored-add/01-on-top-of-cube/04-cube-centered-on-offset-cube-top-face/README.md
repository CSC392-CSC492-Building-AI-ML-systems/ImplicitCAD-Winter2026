# Cube Centered on Offset Cube Top Face Test

## Description
This test validates object-anchored top placement on a cube that is not at the origin and uses decimal coordinates.

## Task
- Start with a base cube at [1.5,-2.25,0.75] with size [3.2,4.6,2.4]
- Add a second cube with the same size centered on the top face of the base cube
- The new cube should be translated to [1.5,-2.25,3.15]

## Key Learning Points
- Object-anchored placement with decimal values
- Top-face centering on non-origin geometry
- Stable z stacking with offset coordinates
