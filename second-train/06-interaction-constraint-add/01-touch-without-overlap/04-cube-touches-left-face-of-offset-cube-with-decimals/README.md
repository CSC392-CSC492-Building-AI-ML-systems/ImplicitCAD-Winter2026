# Cube Touches Left Face Of Offset Cube With Decimals Test

## Description
Validates decimal and off-origin face touching for box primitives.

## Task
- Add a cube of size [1.4,1.2,1.1] that touches the left face of the offset cube with no overlap, centered along y and z on that face.

## Expected Result
The added cube is positioned with exact decimal coordinates and left-face contact.
