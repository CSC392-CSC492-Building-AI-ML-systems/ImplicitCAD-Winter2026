# Cylinder Touches Top Of Offset Cylinder With Decimals Test

## Description
Validates stacked-cylinder contact on off-origin decimal coordinates.

## Task
- Add a cylinder with h=1.8 and r=0.9 directly on top of the existing offset cylinder, sharing the same axis and touching exactly at faces.

## Expected Result
Two cylinders are vertically stacked with exact top/base contact and no overlap.
