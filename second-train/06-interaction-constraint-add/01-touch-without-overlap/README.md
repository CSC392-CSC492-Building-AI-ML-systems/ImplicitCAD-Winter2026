# Touch Without Overlap

## Description
Add one object that is in exact contact with an existing object while avoiding volumetric overlap.

## Cases
- `01-cube-touches-cube-face-no-overlap`
- `02-cylinder-touches-cube-top-face`
- `03-sphere-tangent-to-cylinder-side`
- `04-cube-touches-left-face-of-offset-cube-with-decimals`
- `05-cylinder-touches-top-of-offset-cylinder-with-decimals`
