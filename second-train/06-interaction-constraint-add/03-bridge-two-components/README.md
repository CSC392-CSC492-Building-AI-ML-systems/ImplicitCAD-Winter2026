# Bridge Two Components

## Description
Add one connector object so previously disconnected components become connected by contact.

## Cases
- `01-add-cube-to-connect-2-clusters`
- `02-add-bar-to-connect-2-pillars`
- `03-add-cylinder-bridge-between-two-cylinders`
- `04-add-offset-bar-to-connect-decimal-cuboids`
- `05-add-cylinder-bridge-between-offset-pillars-with-decimals`
