# Cube Between Offset Cuboids Touching Both With Decimals Test

## Description
Validates decimal bilateral face-contact placement between offset cuboids.

## Task
- Add a cuboid of size [3.25,1.4,1.2] between the two offset cuboids so it touches both inner faces, centered in y and z to the shared span.

## Expected Result
A decimal-positioned bridge cuboid satisfies bilateral contact constraints.
