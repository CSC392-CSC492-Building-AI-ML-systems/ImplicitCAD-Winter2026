# Sphere Between Offset Cylinders With Decimal Gaps Test

## Description
Validates centered bilateral gap constraints with asymmetric cylinder radii.

## Task
- Add a sphere of radius 0.8 centered between the inner side surfaces of the two offset cylinders so the sphere has an equal 1.025 unit gap to each cylinder.

## Expected Result
The sphere is off-origin and preserves equal decimal clearances to both anchor cylinders.
