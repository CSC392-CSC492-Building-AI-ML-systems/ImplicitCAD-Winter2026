# Disjoint With Gap

## Description
Add one object that remains separated from existing geometry by a required minimum gap.

## Cases
- `01-sphere-at-least-gap-g-from-any-object`
- `02-cube-at-least-gap-g-from-target-object`
- `03-cylinder-with-gap-from-existing-cylinder-in-x`
- `04-sphere-gap-above-offset-cylinder-top-face`
- `05-cube-gap-from-offset-sphere-along-y`
