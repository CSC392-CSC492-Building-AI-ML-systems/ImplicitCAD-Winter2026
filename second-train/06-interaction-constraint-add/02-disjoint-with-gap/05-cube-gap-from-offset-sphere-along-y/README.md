# Cube Gap From Offset Sphere Along Y Test

## Description
Validates off-origin disjoint placement with decimal gap and mixed signs.

## Task
- Add a cube of size [1.6,2.2,1.2] with a 0.35 unit gap from the existing sphere along +y, centered to the sphere in x and z.

## Expected Result
The cube and sphere remain disjoint with exact y-direction clearance.
